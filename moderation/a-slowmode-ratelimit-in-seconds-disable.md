---
description: >-
  Enables message rate limiting for the channel. Set to 0 or enter "disable" to
  disable slowmode entirely.
---

# a!slowmode \[ratelimit in seconds/"disable"\]

### Examples

```text
a!slowmode 2m
```

```text
a!slowmode 5h
```

```text
a!slowmode 120
```

```text
a!slowmode 0
```

```text
a!slowmode off
```
