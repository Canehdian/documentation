---
description: Gets a users discriminator. It can be anything between 0001 - 9999
---

# user.discriminator <user>

### Examples

{% tabs %}

```text
{user.discriminator}
```
{% endtab %}

{% tab title="Output" %}
```text
1058
```
{% hint style="info" %}
This assumes their tag was Sylver#1058. 
{% endhint %}
