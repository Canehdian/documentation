---
description: Returns all message mentions in a message by ID. This includes user, role and channel mentions.
---
# {message.mentions;[message id];[message channel]}
## Examples
{% tabs %}
{% tab title="Input" %}
```text
{message.mentions}
```
{% endtab %}
{% tab title="Output" %}
```text
111372124383428608 532902669220839426
```
{% endtab %}
{% endtabs %}
