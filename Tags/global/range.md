---
description: Gets a number between [min] and [max].
---

# range [min] [max]

### Examples

{% tabs %}
{% tab title="Input" %}
```text
{range;5;10}
```
{% endtab %}

{% tab title="Output" %}
```text
7
```
{% endtab %}
{% endtabs %}

{% tabs %}
{% tab title="Input" %}
```text
{range;0;100}
```
{% endtab %}

{% tab title="Output" %}
```text
69
```
{% endtab %}
{% endtabs %}
