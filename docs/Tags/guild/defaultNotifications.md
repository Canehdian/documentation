# {guild.defaultNotifications}
## Examples
{% tabs %}
{% tab title="Input" %}
```text
{guild.defaultNotifications}
```
{% endtab %}
{% tab title="Output" %}
```text
Only @mentions
```
{% hint style="info" %}
Output can be either "Only @mentions" or "All Messages".
