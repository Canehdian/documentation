# {user.discriminator;[user]}
## Examples
{% tabs %}
{% tab title="Input" %}
```text
{user.discriminator}
```
{% endtab %}
{% tab title="Output" %}
```text
1058
```
{% endtab %}
{% hint style="info" %}
The users tag in this case would be "Username#1058"
