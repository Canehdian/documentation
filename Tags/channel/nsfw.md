description: Whether or not the current channel is not safe for work.
---

# channel.nsfw <channel\>

### Examples

{% tabs %}
{% tab title="Input" %}

```text
{channel.nsfw}
```

{% endtab %}

{% tab title="Output" %}

```text
false
```

{% hint style="info" %}
This assumes the channel is not marked as NSFW.
{% endhint %}
{% endtab %}
{% endtabs %}
